7 ## What is NeoMutt?
8 
9 * NeoMutt is a project of projects.
10 * A place to gather all the patches against Mutt.
11 * A place for all the developers to gather.
12 
13 Hopefully this will build the community and reduce duplicated effort.
14 
15 NeoMutt was created when Richard Russon (FlatCap) took all the old Mutt patches,
16 sorted through them, fixed them up and documented them.
17 
18 ## What Features does NeoMutt have?
19 
20 | Name | Description
21 |----------------------|-------------------------------------------------------
22 | Attach Headers Color | Color attachment headers using regexp, just like mail bodies
23 | Compose to Sender | Send new mail to the sender of the current mail
24 | Compressed Folders | Read from/write to compressed mailboxes
25 | Conditional Dates | Use rules to choose date format
26 | Encrypt-to-Self | Save a self-encrypted copy of emails
27 | Fmemopen | Replace some temporary files with memory buffers
28 | Forgotten Attachment | Alert user when (s)he forgets to attach a file to an outgoing email.
29 | Global Hooks | Define actions to run globally within Mutt
30 | Ifdef | Conditional config options
31 | Index Color | Custom rules for theming the email index
32 | Initials Expando | Expando for author's initials
33 | Keywords | Labels/Tagging for emails
34 | Kyoto Cabinet | Kyoto Cabinet backend for the header cache
35 | Limit Current Thread | Focus on one Email Thread
36 | LMDB | LMDB backend for the header cache
37 | Multiple FCC | Save multiple copies of outgoing mail
38 | Nested If | Allow complex nested conditions in format strings
39 | New Mail | Execute a command upon the receipt of new mail.
40 | NNTP | Talk to a Usenet news server
41 | Notmuch | Email search engine
42 | Progress Bar | Show a visual progress bar on slow operations
43 | Quasi-Delete | Mark emails that should be hidden, but not deleted
44 | Reply With X-Orig-To | Direct reply to email using X-Original-To header
45 | Sensible Browser | Make the file browser behave
46 | Sidebar | Panel containing list of Mailboxes
47 | Skip Quoted | Leave some context visible
48 | Status Color | Custom rules for theming the status bar
49 | TLS-SNI | Negotiate with a server for a TSL/SSL certificate
50 | Trash Folder | Automatically move deleted emails to a trash bin
