static

collapse/uncollapse all threads

Parameters
menucurrent menu
toggletoggle collapsed state

This function is called by the OP_MAIN_COLLAPSE_ALL command and on folder enter if the OPTCOLLAPSEALL option is set. In the first case, the parameter is 1 to actually toggle collapsed/uncollapsed state on all threads. In the second case, the parameter is 0, actually turning this function into a one-way collapse.
